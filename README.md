# INHA GPA Calculator

Static web app to calculate GPA from INHA University in Tashkent curriculum data.

## Features

- Select faculty/program
- Select semester or level from the top bar
- Use official course names and credits from the attached curriculum files
- Select grades per course
- Calculate weighted GPA, graded credits, total credits, and course count
- Responsive design

## Technologies

- HTML
- CSS
- JavaScript

## Usage

Run a static server from this folder so the browser can load files from `ref/`.

```sh
python3 -m http.server 5173
```

Then open `http://localhost:5173/`.
