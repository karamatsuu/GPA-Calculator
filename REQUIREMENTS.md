# Requirements

## Functional Requirements

- User can select a faculty/program
- User can select a semester, year term, or UK level
- App loads official course names and credits from the attached curriculum files
- User can select grade for each listed course
- App calculates weighted GPA correctly
- App shows graded credits, semester credits, and course count
- User can clear grades for the selected semester

## Non-Functional Requirements

- Clear UI
- Mobile-friendly
- Fast calculation
- Works as a static website
